# CTF-netcat-Checker

Thanks to tr4c3datr4il for this original code and KCTF-pow

A simple QnA-style challenge checker for CTF category with proof-of-work preventing from spamming answer. Use the algorithm like KCTF-pow but without using KCTF :v

Usage: 

`git clone https://github.com/P5ySm1th/CTF-netcat-Checker.git`

`cd CTF-netcat-Checker`

`docker build -t <image_name> .`

`docker run -p 1259:1259 <image_name>`
